### Requirements

- [NodeJS](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) >= 14.8

### Installation

First, make sure Nodejs is running:

```bash
node -v # this should show the version of node
```

Running on the command line

```bash
npm run dev
```

or

```bash
npm start
```
